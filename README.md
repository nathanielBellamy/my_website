# Nate's Website

### build -- Go server, frontend SPA, auth SPA
`MODE=mode ./build.sh`

### build only Go server (fast)
`MODE=mode ./build.sh --server-only`

### serve
`MODE=mode ./serve.sh`

### mode
- `localhost`
- `prod`
- `remotdev`


### Made with: RustWasm, Go, Svelte, Typescript, WebGL, Vite, Tailwind, Flowbite
